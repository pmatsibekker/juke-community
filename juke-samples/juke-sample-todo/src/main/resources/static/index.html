ToDo — record, session-replay, and visible drift detection

The @Juke seam on ToDoDAO.todoService records every call. A cookie session can then replay the same track — and if the UI sends an inconsistent input the per-session report flags the deviation. The controller is also @JukeController, so request/response are captured at the boundary.
